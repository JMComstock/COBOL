# COBOL

Make sure Cobol Compiler installed on your system
```
$ whereis cobc
```
cobc: /usr/bin/cobc /usr/share/man/man1/cobc.1.gz
```
$ which cobc
```
/usr/bin/cobc

If cobol is not installed ...
```
$ sudo apt-get install open-cobol
```

Execute the cobol program
```
$ ./helloworld-exe
```
